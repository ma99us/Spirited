Available API Endpoints

Whisky actions

GET http://ghost.gerdov.com:8181/Spirited/api/whisky/ -> Response getAllWhiskies(int, int, String) - parameters: 'Results Per Page', 'Page Number', 'Sort By'. Returns list of Whisky objects

GET http://ghost.gerdov.com:8181/Spirited/api/whisky/{id} -> Response getWhisky(long) - parameters: Item 'Id'. Returns single Whisky object.

PUT http://ghost.gerdov.com:8181/Spirited/api/whisky/ -> Response createWhisky(Whisky) - parameters: populated 'Whisky' object. Creates a new Whisky object in DB.

POST http://ghost.gerdov.com:8181/Spirited/api/whisky/{id} -> Response updateWhisky(long, Whisky) - parameters: Item 'Id', populated 'Whisky' object. Updates single Whisky object in DB.

DELETE http://ghost.gerdov.com:8181/Spirited/api/whisky/{id} -> Response deleteWhisky(long) - parameters: Item 'Id'. Deletes single Whisky from DB.

GET http://ghost.gerdov.com:8181/Spirited/api/whisky/like/{id} -> Response getSimilarWhisky(long) - parameters: Item 'Id'. Returns list of Whisky objects with similar Flavor Profiles to given one.

Cache maintenance

GET http://ghost.gerdov.com:8181/Spirited/api/cache/status -> Response cacheStatus() - Returns some information and statistics about last time cached data was re-built.

GET http://ghost.gerdov.com:8181/Spirited/api/cache/rebuild -> Response rebuildAllCachedData() - Clears and re-caches all data from third parties. Long operation, could take several minutes!
